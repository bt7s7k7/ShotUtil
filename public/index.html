Drop file or paste screenshot to add image
Use context menu to download image
Q / Delete - Delete
X - Delete All
W - To Front
S - To Back
D - Duplicate
C - Crop Image / Switch color
Shift/Alt + C - Change shape parameter
T - Flip Verically
G - Flip Horizontally
M - Create Ellipse
N - Create Arrow
B - Create Text
V - Create Censor Bar
 Enter / Esc / C - Done
Q - Reset
